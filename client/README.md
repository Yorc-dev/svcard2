# Fuel Flow

Современное веб-приложение для управления потоками данных, построенное на стеке React + TypeScript + Vite с использованием компонентов shadcn/ui.

## Технологический стек

Проект использует передовые инструменты фронтенд-разработки:

- **Framework**: [React 18](https://react.dev)
- **Build Tool**: [Vite](https://vitejs.dev) (для мгновенной сборки и HMR)
- **Language**: [TypeScript](https://www.typescriptlang.org) (строгая типизация)
- **Styling**: [Tailwind CSS](https://tailwindcss.com) + [tailwindcss-animate](https://github.com)
- **UI Components**: [shadcn/ui](https://ui.shadcn.com) (на базе Radix UI primitives)
- **State Management & Data Fetching**: [TanStack Query v5](https://tanstack.com)
- **Forms**: [React Hook Form](https://react-hook-form.com) + [Zod](https://zod.dev) (валидация)
- **Routing**: [React Router DOM v6](https://reactrouter.com)
- **Charts**: [Recharts](https://recharts.org) (визуализация данных)
- **Icons**: [Lucide React](https://lucide.dev)
